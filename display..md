## Definition 

Specifies the display behaviour of an elements rendering box.

## Syntax

```css
display: <value>;
```

The following values can be be appplied to the `display` property: 

- `inline`
- `block`
- `contents`
- `flex`
- `grid`
- `inline-block`
- `inline-flex`
- `inline-grid`
- `list-item`
- `run-in`
- `table`
- `table-caption`
- `table-column-group`
- `table-header-group`
- `table-footer-group`
- `table-row-group`
- `table-cell`
- `table-column`
- `table-row`
- `none`
- `initial`
- `inherit`

## Example 1

Setting a `div` element to behave like a flex box.

``html
<div class="flexdiv">
    <p>First Item</p>
    <p>Second Item</p>
</div>
```

```css
.flexdiv {
    display: flex;
}
```
